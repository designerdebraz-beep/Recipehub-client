# 🍳 Recipe & Food Community Platform

Welcome to the **Recipe & Food Community Platform**! This is a modern, responsive full-stack web application built using **Next.js**, **Tailwind CSS**, and **MongoDB**. Users can explore delicious recipes, like their favorites, view a real-time leaderboard of popular dishes, and report inappropriate content.

## 🚀 Key Features

* **Featured Recipes**: Spotlights handpicked recipes from various global cuisines.
* **Popular Recipes Leaderboard**: Dynamically displays top recipes ranked by their total `likesCount` in a sleek leaderboard grid.
* **Interactive Modals**: Includes functional and responsive popups like the **Report Recipe Modal** to flag spam, copyright, or offensive content.
* **Responsive Grid Layout**: Fully optimized for mobile, tablet, and desktop viewports using custom Tailwind configurations.
* **Dark Mode Support**: Built-in adaptive styling for both light and dark modes.

---

## 🛠️ Tech Stack

### Frontend
* **Framework**: Next.js 14+ (App Router)
* **Styling**: Tailwind CSS
* **Animations**: Framer Motion
* **Icons**: React Icons / SVG Custom Icons

### Backend & Database
* **Runtime**: Node.js & Express.js
* **Database**: MongoDB
* **ODM**: MongoDB Native Driver / Mongoose
